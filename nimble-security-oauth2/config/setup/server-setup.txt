0. Install supervisor
sudo easy_install pip
sudo pip supervisor

Put installation configs in place

1. Place configured jboss-as-7.1.1.Final in /usr/local/
2. Symlink /usr/local/jboss-as-7.1.1.Final /usr/local/jboss
3. Install nimble.sh to /etc/profile.d
4. mkdir /etc/jboss-as; place jboss init.d files (conf and .sh) here
4.1 symlink /etc/init.d/jboss
5. install /etc/supervisord.conf
5.1 install /etc/supervisor/ contents (supervised will look for conf files in subdirs placed here i.e. jboss/XYZ.conf)
6. sudo mkdir -p /var/log/jboss-as
   sudo chown jboss:jboss /var/log/jboss-as

4. Port 8080 will server content to LB
5. Port 9990 will server admin console - should be accessible via remote web browser when on VPN
